关于我们

山东锦禾泓瑞注册地位于山东省潍坊市坊子区，经营范围包括平面设计、互联网销售、技术服务、软件开发、信息系统集成服务等。

旗下公司山东泓瑞钢结构工程有限公司成立于 2019 年 1 月 3 日，专注于钢结构工程、温室工程、集成房屋加工销售等领域，拥有丰富经验和专业团队。
